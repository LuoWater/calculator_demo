###  计算器demo介绍

#### 第一版功能及特点：
1.使用media和em实现简单的响应式布局<br>
2.支持标准的加减乘除四则运算<br>
3.可以进行连续运算<br>
4.使用了HTML5的localStorage进行运算记录的存储，可以进行删除操作

虽然还有很多的bug。。。还在完善当中
